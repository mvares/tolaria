# No-code agencies might be a gateway drug for no-code

Delivering a no/low-code product with an agency might be the **gateway drug** that introduces no-code within the company. 

Later on, **employees could be trained** on Bubble, Webflow, or whatever tool is used, to be able to maintain the applications built this way.
