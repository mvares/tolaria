# ✅ [2020] Start a long term side project 

In order to be long-term-viable, a side project should be built in a way that minimizes the risk of wasting my time.

The best way to do this is building it audience-first. 

## Process

---

1. Find a space you want to build in
1. Create an audience through content
  - I learn about the space
  - Audience will be the first customers
1. Start generating income with info products / freelancing
  - Validate audience needs
1. Build a product for recurring income
## Why is it better?

---

- Each step is low risk
- Each step generates value on its own
- Each step is incremental
  - If you fail one step, you retain the value from the previous ones
