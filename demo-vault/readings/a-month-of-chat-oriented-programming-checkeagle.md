# A Month of Chat-Oriented Programming - CheckEagle

I now believe chat-oriented programming (“CHOP”) can work today, if your tolerance for pain is high enough. ([View Highlight](https://read.readwise.io/read/01kc48v721j0dn89az05kbc6qp))

I embarked on what I call a “month of CHOP” ([View Highlight](https://read.readwise.io/read/01kc48xdys5s30w381v8j5rf3d))

Early in the month of CHOP I let a lot of things go, but over time I have learned it is more productive to stop Claude as soon as I see anything that looks wrong, weird, or dangerous. This is surprisingly stressful, and sometimes I am too late. ([View Highlight](https://read.readwise.io/read/01kc491p5g8bh6ztms4wqt8m1r))

How I work with Claude ([View Highlight](https://read.readwise.io/read/01kc4935af4sxkgpnbsp66g9yh))

There is no question that being able to work with Claude successfully is a different skill from being able to write good code ([View Highlight](https://read.readwise.io/read/01kc493fcqtgj53wee4zwqgj7k))

My current advice for success would be: ([View Highlight](https://read.readwise.io/read/01kc493n5fgs7d7hnb1rsg5x0v))

There is a high cost (stress, head-slapping moments, frustration, token management madness, inventing crazy off-board procedures etc.) But it works (or can work). And it is weirdly addictive, presumably because the highs, when it *does* work well, provide a strong dopamine hit. ([View Highlight](https://read.readwise.io/read/01kc49cy28txy10k4fm9dgb4cs))

To my surprise, I expect to continue using CHOP, probably with Claude Code, for the foreseeable future. I will probably use it differently and a bit less: during the Month of CHOP I specifically wanted to see how far I could get with it doing almost all the writing, but I think a mixed mode will be more likely going forward. I suspect today, a sweet spot for anything complex is for the human to write the outline, structure, and first parts of the code and for the bot to be the critic/pair programmer and finisher/completer. Once the pattern, code style, and testing approach is established, it can fill in the details. But we will see. ([View Highlight](https://read.readwise.io/read/01kc49gp64912vmxknyqm3tgrb))
