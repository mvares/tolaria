# Notes about Circle

## Concepts

- Spaces
  - Members
  - Feed of discussion
  - Different views
    - List View
      - click in order to see the post
    - Card View
    - Post View
  - Different Access Levels
- Groups of spaces
- Totally customizable (look & feel, domain)
- Weekly digest
- Onboarding
- Customize CSS
- Member Tags
  - Totally custom
- Single Sign-On
- Direct Messages
- Powerful Search
- Embeds
  - e.g. Airtable
- Embed Circle
  - Embed whole Circle, or a Space
- Moderation
- Analytics
## Use cases

- Part of the community public and part gated
- Embed parts of Circle in a larger website (e.g. Makerpad)
