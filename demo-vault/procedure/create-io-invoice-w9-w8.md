# Create IO / Invoice / W9 / W8

## 🎯 Goal

For each deal we close, we should send an IO and Invoice to the partner, and make sure they sign it.

## ➡️ Input

- An existing deal that needs an IO or an Invoice to be sent

## ⬅️ Output

- Email sent to the partner with the IO / Invoice

- IO / Invoice stored in the Deals record on Airtablre

## 📋 Instructions

1. Matteo closes a deal

1. Matteo notifies Paco (*who is ideally already in cc in the email thread*), so that he has the info on what is in the deal

1. Paco create the IO / Invoice, and send it to the partner

1. Paco to keep track of the partner replying and ping them partner every three days until done

1. Paco to store the IO / Invoice in the respective Airtable record in the Deals table

1. Pier to keep track of all of these files in dedicated folders for each client’s data @Luca we need to figure out where this repository should be.
