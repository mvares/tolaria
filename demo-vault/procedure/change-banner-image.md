# Change banner image

## 🎯 Goal

- Make sure that newsletters get sent with the correct banner image

## 📋 Instructions

- Open [substack header settings](https://refactoring.fm/publish/settings/preamble)

- Replace banner image according to the daypo

### **1) Every Thursday**

- replace banner image with *podcast banner*

### **2) Every Friday**

- replace banner image with *ideas banner*

### **3) Every Tuesday**

- replace banner image with *essays banner*

## 💻 Tool access

- Substack as an admin
