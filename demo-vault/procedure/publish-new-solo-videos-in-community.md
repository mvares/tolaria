# Publish new solo videos in Community

## 🎯 Goal

- Publish new solo videos in the community library

## ➡️ Input

- New solo video on Youtube

## ⬅️ Output

- Solo video published in the community

## 📋 Instructions

- Create new thread with the video title in the “[Videos](https://community.refactoring.club/c/videos/)” space

- Use [youtube thumbnail](https://www.canva.com/design/DAF233FsV9s/e9KYm1G_dQn6NXQpVAP3AQ/edit) as card view thumbnail

- Put the youtube link inside the thread body

- Change author to Refactoring (from settings)
