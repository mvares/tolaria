# Publish new interviews in Community

## 🎯 Goal

- Publish new podcast interviews in the community

## ➡️ Input

- New podcast interview on Substack

## ⬅️ Output

- Thread published on the community

## 📋 Instructions

- Create card view thumbnail on [Canva file](https://www.canva.com/design/DAGe54ENGxk/OU8pdB9_BtsKyfWKM4WV5A/edit)

- Copy substack summary and paste it as thread body, similarly to [what we do for essays](/1fcbdf02815c800285ccefa5348ee990)

- Change author to Refactoring (from settings)

- Publish inside “Interviews”

<details><summary>If you are migrating an interview from the past, right after publish edit the post to change the timestamp to the correct date</summary></details>
